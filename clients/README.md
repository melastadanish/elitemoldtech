# Client Database

One file per client. Named: `[client-code].md`

Client codes match the quote naming — e.g. `EMT-LM762-Q001` → client code is `LM762`.

## How it works

- When a new client sends files, Claude creates their profile here automatically
- Every new quote updates the `## Quote History` section
- You never need to fill this in manually — Claude does it during quote generation
